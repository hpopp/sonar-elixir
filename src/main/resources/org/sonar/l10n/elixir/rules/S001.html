Public modules should include a @moduledoc attribute describing their purpose.

Noncompliant Code Example

defmodule MyApp.Accounts do
  def list_users do
    Repo.all(User)
  end
end

Compliant Solutions

defmodule MyApp.Accounts do
  @moduledoc """
  Context for managing user accounts.
  """

  def list_users do
    Repo.all(User)
  end
end

To intentionally suppress the warning for internal or private modules:

defmodule MyApp.Accounts.Helpers do
  @moduledoc false

  # ...
end
