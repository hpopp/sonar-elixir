IO.inspect/1 is a debugging tool that writes values to standard output. These calls should be removed before committing to keep logs clean and avoid leaking internal data.

Noncompliant Code Example

def process(data) do
  data
  |> transform()
  |> IO.inspect()
  |> save()
end

Compliant Solution

def process(data) do
  data
  |> transform()
  |> save()
end
