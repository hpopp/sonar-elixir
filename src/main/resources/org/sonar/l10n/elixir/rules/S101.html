Hardcoded credentials in source code can be exploited if the code is ever shared or the repository is compromised.

Noncompliant Code Example

# Hardcoded password
config :my_app, MyApp.Repo,
  password: "s3cret_passw0rd"

# Hardcoded API key
headers = [{"Authorization", "Bearer sk-1234567890abcdef"}]

Compliant Solution

# Use environment variables
config :my_app, MyApp.Repo,
  password: System.get_env("DATABASE_PASSWORD")

# Use runtime configuration
headers = [{"Authorization", "Bearer #{Application.get_env(:my_app, :api_key)}"}]

See

  • CWE-798 - Use of Hard-coded Credentials
  • OWASP Top 10 2021 A7 - Identification and Authentication Failures
