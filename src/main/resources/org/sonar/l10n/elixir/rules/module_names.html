Module names should always be written in PascalCase in Elixir. Each segment of a dotted module name should start with an uppercase letter and contain only letters and digits (no underscores).

Noncompliant Code Example

defmodule MyApp.Web_searchController do
  # ...
end

defmodule MyApp.Foo_bar do
  # ...
end

Compliant Solution

defmodule MyApp.WebSearchController do
  # ...
end

defmodule MyApp.FooBar do
  # ...
end
