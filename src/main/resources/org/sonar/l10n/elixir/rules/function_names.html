Function, macro, and guard names should always be written in snake_case in Elixir. Consistent naming improves readability and follows community conventions.

Noncompliant Code Example

def handleIncomingMessage(message) do
  # ...
end

defp processData(data) do
  # ...
end

defmacro myMacro(arg) do
  # ...
end

Compliant Solution

def handle_incoming_message(message) do
  # ...
end

defp process_data(data) do
  # ...
end

defmacro my_macro(arg) do
  # ...
end
