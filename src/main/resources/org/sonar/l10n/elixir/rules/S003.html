Pipe chains should begin with a raw value — a variable, literal, or data structure — rather than a function call. Starting with a plain value makes the data flow through the pipeline easier to read.

Noncompliant Code Example

String.trim(name) |> String.upcase()

Enum.map(list, &to_string/1) |> Enum.join(", ")

Compliant Solutions

name |> String.trim() |> String.upcase()

list |> Enum.map(&to_string/1) |> Enum.join(", ")
